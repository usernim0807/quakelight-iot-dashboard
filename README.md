# Quakelight IoT Dashboard

An Automatic Emergency Guiding Light and Alert System for  
Cavite State University – Carmona Campus.

## Project Description
Quakelight is an IoT-based system designed to guide occupants during emergency situations using automated guiding lights and real-time alerts.
